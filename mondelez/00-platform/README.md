# 00-platform — Hạ tầng kỹ thuật

Mọi thứ liên quan đến hạ tầng, môi trường và cấu hình kỹ thuật của dự án.

> Nội dung nội bộ kỹ thuật — khách hàng không cần nắm.

## Nội dung

```
00-platform/
├── environments.md     # URL, tenant, env vars theo môi trường
├── deployment.md       # Hướng dẫn deploy, checklist
└── runbook.md          # Vận hành, xử lý sự cố
```

> Clickhouse schema đã chuyển sang `02-data/clickhouse/`
